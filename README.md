# DevOps Automation Scripts

![Bash Linting](https://github.com/serpronichev/devops-auto/actions/workflows/lint.yml/badge.svg)

Репозиторий содержит скрипты автоматизации для администрирования Linux сред (Debian/Ubuntu). Проект автоматизирует рутинные задачи разработчика и разворачивает автономную систему обслуживания логов.

## Описание скриптов
* **setup-env.sh** - Скрипт для первоначальной настройки окружения. Переносит утилиты в систему, создаёт рабочие папки и автоматически настраивает системный планировщик задач.
* **log-cleaner.sh** - Утилита для ротации и очистки старых логов. Находит файлы с расширением `.log` старше заданного количества дней и сжимает их через `gzip`. При развертывании становится доступной как системная команда `/usr/local/bin/log-cleaner`.

## Планировщик задач (Cron)
После запуска `setup-env.sh` в системе регистрируется фоновая задача для пользователя `root`:
* **Расписание:** Каждый день в полночь (00:00).
* **Логика:** Проверка директории `/home/clack92/testlogs` и архивация файлов старше 5 дней.
* **Логирование:** Вывод команд (`stdout`) и ошибки (`stderr`) автоматически перенаправляются и записываются в лог-файл `/var/log/log-cleaner.log`.

## Проверки
В проекте настроен автоматический конвейер **GitHub Actions** (запускается при каждом push-событии):
1. **Линтинг кода:** Статический анализ Bash скриптов с помощью утилиты **ShellCheck**.
2. **Инфраструктурные тесты:** Проверка логики архивации файлов в изолированной среде Ubuntu с помощью тестового фреймворка **BATS** (Bash Automated Testing System).
