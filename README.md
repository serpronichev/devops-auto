# DevOps Automation Scripts

![Bash Lintig](https://github.com/serpronichev/devops-auto/actions/workflows/lint.yml/badge.svg)

Репозиторий содержит скрипты автоматизации для администрирования Linux сред.

## Описание скриптов
* **setup-env.sh** - Скрипт для первоначальной настройки окружения.
* **log-cleaner.sh** - Утилита для ротации и очистки старых логов.


## Проверки
В проекте настроен автоматический линтинг кода с помощью **GitHub Actions** и утилиты **ShellChek**
